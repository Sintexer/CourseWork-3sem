В C ++ есть комментарии в 2 формах: // и / * * /
C ++ использует потоки для вывода на консоль, например cout, и вывод выглядит как cout << "Hello world"
В C ++ переменные должны быть инициализированы перед использованием.
Среди основных типов можно назвать int, double, float, char
char - это один символ в строке, объявление выглядит как char a = 'c';
int - целочисленный тип - int a = 5;
double и float - с плавающей точкой - float a = 5.4;
const - объявление константы - const int a = 5;
Постоянные переменные не могут быть изменены после инициализации
В C ++ функции принимают неограниченное количество аргументов
А деструкторы вообще не принимают аргументов
